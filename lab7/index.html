Lab 7 - Functions

Challenge

The challenge of this lab is to define a function that takes the user's name as a string, converts it into an array, removes spaces, sorts it alphabetically, and then returns it back to the user.

Problems

I accidentally defined a function as "capitalizated" instead of "capitalized" which caused my function to return nothing. However, because I had done the console logs after every step of the function, I was able to easily identify where the issue was and fix it. I also found that the capital letters in the player's name were sorted separately from the lowercase letters, so I had to create a new variable to make it all lowercase before splitting and sorting it.
