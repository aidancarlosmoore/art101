Lab 9 - Libraries & jQuery

Challenge

The challenge of this lab was to get familiar with jQuery, take advantage of its ability to assign multiple aspects in one line rather than having many lines of repetitive code.

Problems

At first, the buttons weren't working because we forgot to actually put an id for each of the buttons in the HTML file. It was trying to ask what was only written as a header initially. Also, I struggled a lot on the bonus task but the hints did help a lot, especially the parent() function, because it made me realize how I could assign them all at once.

Reflection

I put a lot of effort into this assignment, figuring out how to make the buttons but also learning how to assign classes to the buttons and assign the button click outcome to each of them with only one function. I feel more familiar using jQuery in general, as well as doing multiple actions with only one line of code using it.
